Linux — короткий огляд

Односторінкова презентація теми з прикладами команд, Bash-скриптом і джерелами

Автор: Береза Марта

1. Що таке GNU/Linux?

GNU/Linux — це сімейство операційних систем, заснованих на ядрі Linux та інструментах проєкту GNU і численних відкритих програм. Це вільне (open-source) середовище, яке використовується на серверах, десктопах, вбудованих пристроях та суперкомп’ютерах.

2. Популярні дистрибутиви

  • Ubuntu — дружній для користувача десктоп/сервер
  • Debian — стабільність і пакетна інфраструктура
  • Fedora — сучасні технології, швидкі оновлення
  • CentOS Stream / AlmaLinux / Rocky Linux — серверні заміни RHEL
  • Arch Linux — мінімалізм і контроль (rolling-release)
  • openSUSE — потужні адмін-інструменти (yaST)

3. Приклади команд (CLI)

Короткий список команд, що часто використовуються в терміналі:

pwd   # показати поточний робочий каталог
ls    # список файлів і папок
cd    # змінити каталог
echo  # вивести текст
mkdir # створити папку
touch # створити порожній файл
cp    # скопіювати файл або папку
mv    # перемістити або перейменувати
rm    # видалити файл (будьте уважні!)
history # показати історію команд

4. Bash-скрипт: імітація авторизації

Цей простий скрипт демонструє перевірку імені користувача та пароля в текстовому режимі. Це навчальна імітація — не використовуйте для реальної аутентифікації.

# Збережіть як auth.sh і зробіть виконуваним (chmod +x auth.sh)
#!/usr/bin/env bash

# Прості "збережені" облікові дані (для демонстрації)
SAVED_USER="student"
SAVED_PASS="linux123"

read -p "Username: " USER
# -s приховує вивід при введенні пароля
read -s -p "Password: " PASS
echo

if [[ "$USER" == "$SAVED_USER" && "$PASS" == "$SAVED_PASS" ]]; then
  echo "Authentication successful. Welcome, $USER!"
  # тут можна показати меню або інші дії
else
  echo "Authentication failed. Goodbye."
  exit 1
fi

Завантажити auth.sh
